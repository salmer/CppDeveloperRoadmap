# :triangular_ruler: Инструментарий для работы с языком

Начинающие разработчики имеют малый кругозор по доступному инструментарию, который облегчает работу с кодом, повышает эффективность и оберегает от многих ошибок. Все представленные инструменты - не серебряная пуля от всех бед языка, но значительно сглаживают имеющиеся углы. Ниже представлен список распространенных и популярных инструментов, признанных разработчиками по всему миру. Данный список - лишь малая часть доступного инструментария. Со временем вы начнете лучше ориентироваться в них и найдете для себя что-то новое.

## :gear: Компиляторы

* :arrow_forward: **GCC (GNU Compiler Collection)**

    Сайт: [https://gcc.gnu.org](https://gcc.gnu.org)

    Стоимость: бесплатно

    Компилятор по умолчанию в большинстве дистрибутивов Linux и один из самых распространенных компиляторов C++ в мире. Оперативно поддерживает новейшие стандарты языка и генерирует хорошо оптимизированный код. Под Windows доступен через проекты MSYS2 и MinGW-w64.

* :arrow_forward: **Clang**

    Сайт: [https://clang.llvm.org](https://clang.llvm.org)

    Стоимость: бесплатно

    Компилятор, построенный на инфраструктуре LLVM, компилятор по умолчанию в macOS. Известен быстрой компиляцией и понятными, дружелюбными сообщениями об ошибках, что делает его хорошим выбором для новичков. На базе Clang построено целое семейство инструментов разработчика: clang-format, clang-tidy, а также языковой сервер clangd, используемый многими редакторами.

* :arrow_forward: **MSVC (Microsoft Visual C++)**

    Сайт: [https://visualstudio.microsoft.com](https://visualstudio.microsoft.com)

    Стоимость: входит в состав Visual Studio (Community Edition - бесплатно)

    Компилятор от Microsoft, поставляется в составе Visual Studio. Основной выбор для разработки под Windows: большинство коммерческого ПО и игровых проектов под Windows собираются именно им.

    **Совет:** Быстро проверить свой код сразу на всех трех компиляторах, ничего не устанавливая, можно с помощью [Compiler Explorer](https://godbolt.org).

## :page_facing_up: Текстовые редакторы

* :arrow_forward: **Visual Studio Code**

    Сайт: [https://code.visualstudio.com/](https://code.visualstudio.com/)

    Стоимость: бесплатно

    Мощный и эффективный редактор текстовых файлов и исходного кода. Имеет богатую библиотеку расширений, которая позволит настроить его под себя. Также возможно настроить его под работу с исходным кодом: компиляция, запуск и отладка. Обладает мощным поисковым движком, по файлам и папкам, что повышает эффективность поиска, чтения и работы с большими проектами.


* :arrow_forward: **Notepad++**

    Сайт: [https://notepad-plus-plus.org/](https://notepad-plus-plus.org/)

    Стоимость: бесплатно

    Легковесный редактор текстовых файлов и исходного кода. Поддерживает синтаксис и подсветку распространенных языков программирования. По сравнению с Visual Studio Code, его удобно использовать для быстрого открытия и просмотра файлов. За счет своей легковесности комфортно работать с большим количеством текстовых файлов.


## :open_file_folder: IDE (Integrated Development Environment)

* :arrow_forward: **Microsoft Visual Studio IDE**

    Сайт: [https://visualstudio.microsoft.com](https://visualstudio.microsoft.com)

    Стоимость: Community Edition - бесплатно

    Интегрированная среда разработки от компании Microsoft. Предоставляет весь необходимый набор инструментов (редактор кода, компилятор, отладчик, профилировщик и т.д.) из коробки. Поддерживает разработку на различных языках программирования, а также кроссплатформенную разработку. Отличное начало для новичков, т.к. современный интерфейс студии максимально дружелюбен, и практически не требует какой-либо доработки из коробки.


* :arrow_forward: **Qt Creator IDE**

    Сайт: [https://www.qt.io/product/development-tools](https://www.qt.io/product/development-tools)
    
    Стоимость: бесплатно в open source проектах (более подробно: [Qt Open Source](https://www.qt.io/download-open-source?hsCtaTracking=9f6a2170-a938-42df-a8e2-a9f0b1d6cdce%7C6cb0de4f-9bb5-4778-ab02-bfb62735f3e5))

    Изначально Qt Creator позиционировался как IDE для разработки графических интерфейсов приложений, разрабатываемых на языке C++. Со временем фреймворк оброс огромными возможностями. Все переросло в полноценную экосистему для разработки кроссплатформенных приложений. Фреймворк предоставляет большую библиотеку примитивов для различных потребностей: работа с сетью, графический интерфейс, работа с базами данных, работа с популярными форматами: изображения, текстовые файлы и т.д. Современный Qt Creator выступает в роли конкурента для Visual Studio, но в основном он снискал славу в среде разработчиков, которые разрабатывают приложения под различные дистрибутивы Linux.


* :arrow_forward: **Xcode**

    Сайт: [https://developer.apple.com/xcode/](https://developer.apple.com/xcode/)

    Стоимость: бесплатно

    Штатная среда разработки Apple и единственный способ собирать приложения под macOS и iOS. Поставляется вместе с Clang и LLDB. Если вы работаете под macOS, Xcode понадобится как минимум ради инструментов командной строки, даже если писать код вы предпочитаете в другом редакторе.


* :arrow_forward: **JetBrains CLion IDE**

    Сайт: [https://www.jetbrains.com/clion](https://www.jetbrains.com/clion)

    Стоимость: бесплатно для некоммерческого использования (с мая 2025 года), для коммерческой разработки - платная

    Мощная мультиплатформенная IDE от компании JetBrains. Как и другие IDE, она содержит полный набор инструментов для комфортной разработки программного обеспечения. Удобен для кроссплатформенной разработки как на Cи, так и на C++.

Является конкурентом Visual Assist.

    Сайт: [https://www.wholetomato.com](https://www.wholetomato.com)

    Расширение для MS Visual Studio. Добавляет дополнительные возможности для работы с исходным кодом: расширенная подсветка кода и подсказки, подробная информация во время отладки или при написании кода, продвинутый поиск, навигация по проектам и т.д. Является конкурентом JetBrains ReSharper.


* :arrow_forward: **Incredibuild**

    Сайт: [https://www.incredibuild.com](https://www.incredibuild.com)

    Приложение/расширение для распределенной сборки проектов. Объединяет рабочие станции разработчиков в единую сеть, за счет чего сборка исходного кода может производиться одновременно на десятках машин параллельно. Это позволяет ускорить скорость сборки больших проектов в несколько раз.


## :electric_plug: Пакетные менеджеры и системы сборки

* :arrow_forward: **CMake**

    Сайт: [https://cmake.org](https://cmake.org)

    Кроссплатформенная система автоматизации сборки приложения из исходного кода. Генерирует необходимые артефакты для последующей сборки приложения на целевой платформе. На текущий момент считается "стандартным" инструментом для сборки различных библиотек, в случае поставки в качестве исходного кода.

* :arrow_forward: **Conan**

    Сайт: [https://conan.io](https://conan.io)

    Стоимость: бесплатно

    Пакетный менеджер, а также менеджер зависимостей для организации C++ библиотек и фреймворков. Поддерживает работу с различными платформами: Windows, Linux, etc. Поддерживает интеграцию с CMake, Visual Studio и т.д.


* :arrow_forward: **vcpkg**

    Сайт: [https://vcpkg.io](https://vcpkg.io)

    Стоимость: бесплатно

    Бесплатный пакетный менеджер с открытым исходным кодом для библиотек C и C++ от компании Microsoft. Предоставляет тысячи готовых к использованию библиотек и легко интегрируется с CMake и Visual Studio. Наряду с Conan является одним из двух самых популярных менеджеров зависимостей в экосистеме C++.


* :arrow_forward: **Ninja**

    Сайт: [https://ninja-build.org](https://ninja-build.org)

    Стоимость: бесплатно

    Менеджер сборки проектов для приложений, написанных на Си и C++. Основное преимуществ менеджера: быстрая сборка проектов. Поддерживает кроссплатформенную разработку, поддерживает все популярные компиляторы.

* :arrow_forward: **ccache**

    Сайт: [https://ccache.dev](https://ccache.dev)

    Стоимость: бесплатно

    Кэш компилятора: он сохраняет результаты предыдущих компиляций и повторно использует их при повторной компиляции того же кода, что может значительно ускорить пересборку. Он прозрачно работает поверх GCC или Clang и интегрируется с CMake парой строк кода, что делает его одним из самых простых способов ускорения сборки в средних и крупных проектах.


## :mag: Анализаторы и форматтеры кода

* :arrow_forward: **clang-format**

    Сайт: [https://clang.llvm.org/docs/ClangFormat.html](https://clang.llvm.org/docs/ClangFormat.html)

    Стоимость: бесплатно

    Де-факто стандартный автоматический форматтер кода для C++. Единый файл конфигурации (`.clang-format`) в репозитории поддерживает единообразный стиль кода всей команды и избавляет от споров о форматировании на ревью кода. Поддерживается из коробки практически всеми редакторами и IDE.

* :arrow_forward: **clang-tidy**

    Сайт: [https://clang.llvm.org/extra/clang-tidy/](https://clang.llvm.org/extra/clang-tidy/)

    Стоимость: бесплатно

    Инструмент статического анализа (линтер) из проекта LLVM. Проверяет код на соответствие сотням правил: подверженные ошибкам конструкции, производительность, читаемость, современный стиль C++. Многие найденные проблемы умеет исправлять автоматически. Интегрирован в большинство IDE: Visual Studio, CLion, VS Code, Qt Creator.

* :arrow_forward: **Санитайзеры (ASan, UBSan, TSan)**

    Сайт: [https://github.com/google/sanitizers](https://github.com/google/sanitizers)

    Стоимость: бесплатно

    Инструменты динамического анализа, встроенные непосредственно в GCC, Clang и MSVC и включаемые флагами компилятора (например, `-fsanitize=address`). AddressSanitizer ловит ошибки работы с памятью (выход за границы, use-after-free), UndefinedBehaviorSanitizer - неопределенное поведение, ThreadSanitizer - гонки данных. Запуск тестов с включенными санитайзерами считается базовой практикой современной разработки на C++.

* :arrow_forward: **Фаззинг (libFuzzer, AFL++)**

    Сайт: [https://llvm.org/docs/LibFuzzer.html](https://llvm.org/docs/LibFuzzer.html), [https://github.com/AFLplusplus/AFLplusplus](https://github.com/AFLplusplus/AFLplusplus)

    Стоимость: бесплатно

    Фаззинг - это автоматическая подача в код потока случайных и намеренно некорректных данных в поисках падений и зависаний. Особенно полезен для всего, что разбирает внешний ввод: парсеров форматов, сетевых протоколов, декодеров. Обычно запускается вместе с санитайзерами - так фаззер находит ввод, приводящий к ошибке, а санитайзер показывает, где именно она произошла. libFuzzer встроен в Clang, AFL++ работает как самостоятельный инструмент.

* :arrow_forward: **PVS Studio**

    Сайт: [https://pvs-studio.com](https://pvs-studio.com)

    Стоимость: платно; бесплатно для open-source проектов и студентов

    Кроссплатформенный (Windows, Linux, MacOS) статический анализатор кода от российской компании PVS-Studio. Основная задача анализатора - провести анализ исходников на предмет различных ошибок, которые не обнаруживаются компиляторами или на этапе ревью кода. Благодаря ему можно минимизировать ошибки, связанные с синтаксическими конструкциями языка и их подводными камнями.


* :arrow_forward: **Cppcheck**

    Сайт: [https://cppcheck.sourceforge.io](https://cppcheck.sourceforge.io)

    Стоимость: бесплатно

    Бесплатный анализатор кода. Поможет отыскать распространенные ошибки при помощи статического анализа исходного кода, которые могут быть упущены компилятором или разработчикам. Кроссплатформенный, поддерживает популярные дистрибутивы Linux, работает под Windows.


* :arrow_forward: **Valgrind**

    Сайт: [https://valgrind.org](https://valgrind.org)

    Стоимость: бесплатно

    Набор инструментов, который поможет исследовать разнообразные проблемы во время работы приложения: утечка памяти, профилирование тормозов и т.д. Заточен для работы с различными дистрибутивами Linux. В современных проектах многие его задачи решаются санитайзерами с гораздо меньшими накладными расходами, но Valgrind остается полезен тем, что не требует перекомпиляции.

## :beetle: Отладчики

* :arrow_forward: **GDB (GNU Debugger)**

    Сайт: [https://sourceware.org/gdb/](https://sourceware.org/gdb/)

    Стоимость: бесплатно

    Стандартный отладчик в мире GNU/Linux. Позволяет ставить точки останова, выполнять код по шагам, просматривать переменные и память, анализировать дампы упавших приложений (core dumps). Большинство графических отладчиков в IDE под Linux (VS Code, Qt Creator, CLion) используют GDB под капотом, поэтому знание его основ пригодится, даже если вы в основном отлаживаете из IDE.

* :arrow_forward: **LLDB**

    Сайт: [https://lldb.llvm.org](https://lldb.llvm.org)

    Стоимость: бесплатно

    Отладчик из проекта LLVM и отладчик по умолчанию в macOS (используется в Xcode). По возможностям близок к GDB, но имеет более современную архитектуру.

* :arrow_forward: **WinDbg**

    Сайт: [https://learn.microsoft.com/windows-hardware/drivers/debugger/](https://learn.microsoft.com/windows-hardware/drivers/debugger/)

    Стоимость: бесплатно

    Отладчик от Microsoft для Windows. В повседневной работе под Windows обычно достаточно отладчика, встроенного в Visual Studio, но WinDbg незаменим там, где его возможностей не хватает: разбор аварийных дампов с боевых машин, отладка драйверов и режима ядра, работа с приложением без исходников. Полезный навык для тех, кто занимается низкоуровневой разработкой или разбором падений у пользователей.

## :stopwatch: Профилировщики

* :arrow_forward: **perf**

    Сайт: [https://perfwiki.github.io/main/](https://perfwiki.github.io/main/)

    Стоимость: бесплатно

    Стандартный сэмплирующий профилировщик в Linux, встроенный в ядро. Он показывает, на что приложение фактически тратит процессорное время, с детализацией аппаратных счетчиков (промахи кэша, ошибки предсказания переходов), когда это необходимо. Часто используется вместе с визуализацией flame-graph, чтобы сделать узкие места (hotspots) очевидными.

* :arrow_forward: **Tracy**

    Сайт: [https://github.com/wolfpld/tracy](https://github.com/wolfpld/tracy)

    Стоимость: бесплатно

    Профилировщик кадров в реальном времени, особенно популярный в разработке игр. Вы размечаете зоны в своем коде легковесными макросами и наблюдаете за таймингами в реальном времени в графическом клиенте, вплоть до отдельных кадров и потоков. Кроссплатформенный инструмент с удивительно низкими накладными расходами.

* :arrow_forward: **Intel VTune Profiler**

    Сайт: [https://www.intel.com/content/www/us/en/developer/tools/oneapi/vtune-profiler.html](https://www.intel.com/content/www/us/en/developer/tools/oneapi/vtune-profiler.html)

    Стоимость: бесплатно

    Мощный профилировщик для глубокого анализа производительности на архитектуре x86: узкие места, эффективность многопоточности, паттерны доступа к памяти и метрики уровня микроархитектуры. Инструмент первого выбора, когда вывода `perf` недостаточно. В Windows встроенный в Visual Studio IDE профилировщик процессора и памяти также предоставляет хорошие возможности.

## :white_check_mark: Тестирование и бенчмарки

* :arrow_forward: **GoogleTest (gtest/gmock)**

    Сайт: [https://github.com/google/googletest](https://github.com/google/googletest)

    Стоимость: бесплатно

    Самый распространённый фреймворк юнит-тестирования в C++. В комплекте идёт GoogleMock для создания заглушек и моков, что позволяет тестировать код в отрыве от его зависимостей - базы данных, сети, файловой системы. Хорошо интегрируется с CMake и подключается через любой из популярных пакетных менеджеров.

* :arrow_forward: **Catch2**

    Сайт: [https://github.com/catchorg/Catch2](https://github.com/catchorg/Catch2)

    Стоимость: бесплатно

    Более лёгкая альтернатива GoogleTest с лаконичным синтаксисом. Тест здесь - это обычная функция с макросом `REQUIRE` вместо набора специальных макросов сравнения. Хороший выбор для небольших проектов и для знакомства с юнит-тестированием: подключается буквально одним заголовочным файлом.

* :arrow_forward: **Google Benchmark**

    Сайт: [https://github.com/google/benchmark](https://github.com/google/benchmark)

    Стоимость: бесплатно

    Библиотека для микробенчмарков. Сама подбирает число повторений, чтобы результат был статистически значимым, и умеет бороться с тем, что оптимизатор выбрасывает код, результат которого никуда не используется. Нужна там, где требуется сравнить две реализации по скорости: замер "вручную" через замер времени до и после почти всегда даёт недостоверный результат.

## :robot: AI-инструменты

AI-ассистенты стали частью повседневного инструментария разработчика: они ускоряют написание шаблонного кода, помогают разобраться в незнакомой кодовой базе и объясняют ошибки компилятора. При этом они не заменяют знание языка - ниже расписано, почему.

* :arrow_forward: **GitHub Copilot**

    Сайт: [https://github.com/features/copilot](https://github.com/features/copilot)

    Стоимость: есть бесплатный тариф с ограничениями; полный доступ - платно, бесплатно для студентов, преподавателей и мейнтейнеров популярных open source проектов

    Дополнение кода прямо в редакторе: подсказывает следующие строки по контексту файла и проекта. Интегрируется с Visual Studio, VS Code, CLion и другими популярными IDE.

* :arrow_forward: **Claude Code**

    Сайт: [https://claude.com/claude-code](https://claude.com/claude-code)

    Стоимость: платно, в рамках подписки

    Ассистент, работающий в терминале и в IDE: читает проект целиком, вносит правки в несколько файлов сразу, запускает сборку и тесты. Ориентирован на задачи крупнее одной строки - рефакторинг, разбор незнакомого кода, написание тестов.

* :arrow_forward: **Cursor**

    Сайт: [https://cursor.com](https://cursor.com)

    Стоимость: есть бесплатный тариф, расширенные возможности - платно

    Редактор на базе VS Code со встроенным AI-ассистентом. Умеет отвечать на вопросы по кодовой базе и вносить правки сразу в нескольких файлах, сохраняя привычные расширения и настройки VS Code.

* :arrow_forward: **Локальные модели (Ollama, llama.cpp)**

    Сайт: [https://ollama.com](https://ollama.com), [https://github.com/ggml-org/llama.cpp](https://github.com/ggml-org/llama.cpp)

    Стоимость: бесплатно

    Запуск моделей на своей машине. Уступают облачным по качеству и требуют заметных ресурсов, зато код не покидает вашего компьютера. Вариант для проектов, где отправка исходников во внешние сервисы запрещена. Кстати, `llama.cpp` - сам по себе показательный пример современного C++ проекта.

### :warning: О чём стоит помнить

- **Проверяйте всё, что сгенерировано.** Модель может выдать правдоподобный, но неверный код: несуществующие функции стандартной библиотеки, тонкие ошибки времени жизни объектов, состояния гонки. В C++ цена такой ошибки высока - неопределённое поведение может не проявиться ни на одном тесте, а выстрелить у пользователя. Именно поэтому знание языка остаётся обязательным: чтобы проверить ответ, нужно понимать предмет не хуже, чем при написании кода вручную.
- **Соблюдайте политику компании.** Во многих организациях отправка рабочего кода во внешние сервисы ограничена или запрещена. Уточните правила до того, как подключать ассистента, а не после.
- **Помните про лицензии.** Сгенерированный код может воспроизводить фрагменты чужих проектов вместе с их лицензионными обязательствами. Для коммерческих проектов это отдельный риск, который стоит обсудить с командой.
- **Не пропускайте этап обучения.** Ассистент избавляет от рутины, но если он решает за вас задачи, на которых вы учитесь, - вы не вырастете как инженер. На старте карьеры полезно сначала решить задачу самому, а потом сравнить с тем, что предложит модель.

## :floppy_disk: Git клиенты

* :arrow_forward: **SmartGit**

    Сайт: [https://www.syntevo.com/smartgit/](https://www.syntevo.com/smartgit/)

    Стоимость: бесплатная для личных или open source проектов, в ином случае - платная

    Полноценный кроссплатформенный комбайн для работы с git репозиториями. Из коробки предоставляет следующие возможности: прием/отправка изменений в репозитории, просмотр истории изменений, текстовый редактор для разрешения конфликтов и т.д. Поддерживает интеграцию со всеми популярными репозиториями: GitHub, BitBucket, GitLab и т.д.

* :arrow_forward: **Atlassian SourceTree**

    Сайт: [https://www.sourcetreeapp.com/](https://www.sourcetreeapp.com/)

    Стоимость: бесплатно

    Отличная бесплатная альтернатива для работы с git через графический интерфейс. Не уступает по функционалу SmartGit, за исключением отсутствия собственного редактора разрешения конфликтов. Это легко исправляется интеграцией с Visual Studio Code или любым другим редактором, который умеет сравнивать файлы между собой. Обратите внимание: в отличие от SmartGit, доступен только под Windows и macOS. Поддерживает интеграцию с публичными репозиториями: GitHub, BitBucket, GitLab и т.д.


* :arrow_forward: **Git Kraken**

    Сайт: [https://www.gitkraken.com/](https://www.gitkraken.com/)

    Стоимость: бесплатная для личных или open-source проектов, в ином случае - платная

    Кроссплатформенный и высокоэффективный клиент для Windows, Linux, MacOS. Поддерживает интеграцию с GitHub, Bitbucket и Gitlab, а также весь необходимый функционал для повседневной работы: изучение истории изменений, прием/отправка изменений, переключение между ветками, встроенный редактор разрешения конфликтов и т.д.
